## 1. 产品概述

### 1.1 产品愿景
打造一款完全开源、免费、跨平台的浏览器流媒体下载扩展，让用户以最低技术门槛获取 HLS 和 DASH 视频。

### 1.2 核心价值主张
- **双格式支持**：同时完整解析 M3U8 和 MPD，市场唯一
- **零依赖**：浏览器内 FFmpeg.wasm 转封装，无需装软件
- **完全免费**：无付费墙、无水印、无广告、本地运行保护隐私
- **智能高效**：32 线程并发、自动解密、断点续传

## 2. 功能需求

### 2.1 功能分级

**P0（V1.0 必做）**
1. 网络请求嗅探引擎（URL后缀 + Content-Type 双重识别）

4. 下载管理（进度、暂停/恢复/取消、下载列表）
5. 多线程并发（默认 6 线程，可调）

**P1（V1.1 高优先级）**

8. 直播流录制（持续刷新+实时合并）
9. 视频预览
10. 下载队列与批量管理
11. 侧边栏模式
12. 格式选择（分辨率、音轨多选）

**P2（V1.2 计划中）**

16. 自定义下载规则（正则/域名黑白名单）

19. 多语言界面
20. 开发者调试模式

### 2.2 核心功能详细说明

#### 网络嗅探引擎
- 使用 `chrome.webRequest` (MV3 观察模式) + `declarativeNetRequest`
- 识别 18 种媒体类型，自动去重、分类显示（视频/音频/图片/其他）
- 支持按域名、大小过滤

#### M3U8 解析与下载
- 解析 Master/Media Playlist
- 提取 `#EXT-X-KEY` 加密信息
- 支持相对/绝对路径拼接
- 多线程下载 TS，二进制拼接为视频文件

#### MPD 解析与下载（差异化核心）
- 解析 Period → AdaptationSet → Representation 层级
- 支持 SegmentTemplate（$Number$/$Time$）和 SegmentList
- 支持字节范围请求
- 下载初始化段和所有媒体段，FFmpeg.wasm 混流

#### 加密处理
- 自动下载密钥并解密 AES-128 CBC/ECB
- 支持手动输入十六进制或 Base64 密钥
- IV 自定义偏移配置

#### 直播录制
- 持续刷新播放列表，下载新增分片
- 实时合并，检测停播或手动停止

#### 用户界面
- 弹出面板 / 侧边栏 / 独立解析器页面
- 暗色模式
- 视频缩略图预览

## 3. 非功能需求

### 3.1 性能指标
- 资源检测响应 ≤ 0.3s
- M3U8/MPD 解析 ≤ 0.5s
- 后台内存 ≤ 30MB，CPU ≤ 3%
- 首屏加载 ≤ 1s
- 扩展包体 ≤ 2MB（不含按需加载的 FFmpeg.wasm）

### 3.2 兼容性
- Chrome/Edge 93+（推荐 104+）
- Firefox 最新版（需非国区 IP）
- 移动端 Kiwi Browser / Edge Android

### 3.3 安全与隐私
- 所有操作全本地完成
- 零数据收集、代码开源、权限最小化
- 网络请求全 HTTPS
