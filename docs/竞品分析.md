# 浏览器 M3U8 / MPD 视频下载插件 — 竞品分析报告

## 1. 报告概述

### 1.1 分析目的
为开发一款全新的浏览器端 M3U8（HLS）与 MPD（DASH）视频下载插件提供市场调研、技术分析、竞品拆解和产品策略建议。

### 1.2 技术背景
现代在线视频平台普遍采用流媒体传输。HLS 通过 M3U8 文件将视频切分为 TS 分片，支持多码率自适应。DASH 则使用 MPD 清单描述分片信息，格式为 XML。两种协议均基于 HTTP，方便 CDN 分发。

### 1.3 用户痛点
- 资源定位困难：78% 用户找不到真实视频链接
- 流媒体格式复杂：M3U8/MPD 分段传输无法直接保存
- 批量效率低下：手动下载耗时是工具的 4.2 倍
- 加密/签名链接增加下载难度

## 3. 重点竞品深度分析

### 3.1 猫抓 (Cat-Catch) — 行业标杆
- **定位**：开源资源嗅探扩展，面向技术用户
- **技术架构**：webRequest 捕获 → M3U8/MPD 解析 → 多线程下载器
- **优势**：18 种格式支持、AES-128 解密、CPU/内存占用极低、可联动 N_m3u8DL-RE
- **劣势**：YouTube 主动屏蔽、Firefox 国区受限、新手门槛高

### 3.2 Stream Detector — 多协议专家
- **核心能力**：通过 Content-Type+扩展名双重校验识别流媒体，支持 HLS/DASH/HDS
- **特色**：内存占用<50MB，可导出任务至 yt-dlp/FFmpeg，全本地运行

### 3.3 Video DownloadHelper — 商业化困局
- 强制水印、付费墙、仅 2.1/5 的 Trustpilot 评分
- **启示**：开源+本地运行的免费模式更受信任

### 3.4 N_m3u8DL-RE — 桌面端最强搭档
- 支持实时合并直播、多线程下载、自定义音视频轨道选择、AES-128 解密

## 4. 技术可行性分析

### 4.1 下载链路

**HLS (M3U8)**
1. 拦截 M3U8 URL
2. 解析媒体分片地址和加密密钥
3. 多线程下载 TS 分片
4. AES 解密（如有）
5. 合并为 MP4

**DASH (MPD)**
1. 拦截 MPD 文件
2. 解析 Period/AdaptationSet/Representation
3. 根据 SegmentTemplate 或 SegmentList 获取分片地址
4. 下载初始化段和媒体段 (.m4s)
5. 使用 FFmpeg.wasm 混流

### 4.2 Chrome Extension 技术栈
- Manifest V3：必须使用 declarativeNetRequest 或观察模式 webRequest
- 核心 API：Downloads、Storage、Tabs、notifications、sidePanel
- FFmpeg.wasm：浏览器内完成转封装，无需本地安装

### 4.3 平台对比

| 维度 | Chrome/Edge | Firefox | Safari |
|------|------------|---------|--------|
| 扩展规范 | Manifest V3 | MV2/MV3 | Web Extensions |
| webRequest | 仅观察 | 完整支持 | 有限 |
| Downloads API | ✅ | ✅ | ❌ |
| WebAssembly | ✅ | ✅ | ✅ |

## 5. 市场机会分析

### 5.1 差异化机会
- **MPD 完整解析下载**：当前浏览器扩展几乎一致缺失
- **完全免费开源**：无付费、无水印、无广告
- **浏览器内转封装**：FFmpeg.wasm 零依赖
- **移动端适配**：支持 Kiwi/Edge Android
- **直播录制/断点续传**：功能补全

### 5.2 目标用户画像
- 普通用户（60%）：一键保存课程、社交媒体视频
- 内容创作者（20%）：直播素材采集、批量下载
- 技术爱好者（20%）：高级配置、命令行集成

### 5.3 合规风险
- 商店下架风险：参考猫抓主动屏蔽 YouTube
- DRM 限制：产品定位为非 DRM 下载工具
- 免责声明：提示用户遵守当地版权法

## 6. 竞争策略总结
- **核心差异化**：MPD/DASH 全流程 + 开源免费
- **架构创新**：浏览器内 FFmpeg.wasm 混流
- **体验创新**：现代 UI、暗色模式、视频预览、拖拽批量
- **生态创新**：灵活导出至 N_m3u8DL-RE/FFmpeg/yt-dlp
